WatPark - A WP8 App done in one night

WP8, WatPark, UWaterloo, OpenData

Download this project as a .zip file Download this project as a tar.gz file

WatPark uses Open Data API developed by students of University of Waterloo to display parking lots information.

Main Features

  • List parking lots around the university
  • Retrieve real-time number of available parking lots information from api.uwaterloo.ca
  • Done within one night, a try out of WP8 app development

To clone this project:

$ git clone https://github.com/gyfelton/WP8-WatPark.git
